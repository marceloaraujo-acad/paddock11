# paddock11
PADDOCK 11 é uma aplicação web criada para fins educacionais, é de página única que gerencia um registro da equipe de um time de Fórmula 1 - pilotos, engenheiros e integrantes do box. A interface consome em tempo real uma API REST publicada no Render e permite as operações completas de um CRUD (criar, listar, editar e remover).
